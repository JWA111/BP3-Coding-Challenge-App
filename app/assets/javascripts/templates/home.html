Application Details

Task Overview

Use this page to generate reports displaying counts of open and closed tasks given constraints.
Options are:

  • System State

    Given a date, report the state of the system at that point in time.

  • Time Span

    Given a date interval, report the count of tasks open during the interval as well as tasks closed during the interval.

  • Instance

    Given an Instance ID, report the count of all tasks currently open and closed associated with the instance.

  • Assignee

    Given the name of an assignee, report the count of all tasks currently open and closed associated with the assignee.

Task Search

Use this page to search for tasks given filters.
filters are:

  • Search String

    Type a value associated with the task you are searching for

  • Date Intervals

    Takes one or two dates. If only one date is provided the results are found similarly to the system state report.
    If two dates are given, the search criteria is constrained to the time interval.
    Note that if the task status is not specified, the search will look for tasks that were both opened and closed during the interval

  • Instance Attributes

    Allows the use of filters related to the instance. Also allows you to specify the state of the task.
    The state filter only takes "open" or "closed".

  • Assignee

    Takes the name of an assignee, only tasks related to the specified assignee will be found.

  • Parent Process

    Takes the name of a process, only tasks related to that process will be included.
